# Task 1: AI-Powered Sales Forecasting

## Objective
To analyze historical sales data and forecast future sales trends to help
businesses make data-driven decisions.

## Description
This project uses past sales data to calculate a 7-day moving average and
predict future sales values. The results are visualized using line graphs
for better understanding of trends.

## Files in this Repository
- `Sales_Forecasting_Task1.ipynb` – Jupyter notebook with full code
- `sales.csv` – Dataset containing historical sales data
- `sales_forecast_with_future.png` – Output graph with future prediction

## Tools Used
- Python
- Pandas
- Matplotlib

## Outcome
Successfully generated a future sales forecast based on historical data.
